# Shiftcore Certificatenchecker v2

Interne, statische webapp voor het controleren van functies, certificaten en aanwijzingen in de infra.

## Functionaliteit
- Premium Shiftcore zwart/geel dashboard
- Sectorfilters en zoekfunctie
- Klikbare functiekaarten
- Functiedetails in een popup
- Certificaatuitleg en bronnen
- Tijdelijke certificatenchecker zonder persoonsgegevens of opslag
- Responsive voor desktop en mobiel

## Installatie in GitHub
Verwijder de oude bestanden en upload de losse bestanden uit deze ZIP rechtstreeks naar de hoofdmap van de repository.

## Data aanpassen
Alle functies en certificaten staan in `data.js`.

## Belangrijk
Deze tool geeft een praktisch intern overzicht. Een functienaam is niet automatisch gelijk aan een formele aanwijzing. Controleer altijd:
- exacte werkzaamheden;
- juiste BEI/VIAG/KIAD-verbijzondering;
- geldig persoonscertificaat;
- schriftelijke aanwijzing van de werkgever;
- eisen van opdrachtgever en netbeheerder;
- bedrijfsspecifieke instructies.
