# Shiftcore Certificatenchecker

Interne webapp om per sector en functie te controleren welke certificaten nodig zijn.

## Gebruik

Open `index.html` lokaal, of upload alle bestanden naar de hoofdmap van de GitHub-repository.

## Vercel

- Framework preset: Other
- Build command: leeg
- Output directory: leeg
- Install command: leeg

## Bestanden

- `index.html` – structuur
- `styles.css` – Shiftcore huisstijl
- `data.js` – functies en certificaten
- `app.js` – filters en functievenster
- `shiftcore-logo.png` – Shiftcore logo
- `vercel.json` – Vercel configuratie

## Inhoud wijzigen

Nieuwe functies en certificaten kunnen rechtstreeks in `data.js` worden toegevoegd.

De getoonde eisen zijn indicatief. Controleer altijd de definitieve eisen van opdrachtgever, netbeheerder, project en werkzaamheden.
